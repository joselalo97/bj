Introducción

El diseño de una plataforma web condiciona en gran medida cómo navegan e interactúan los usuarios; por esa razón, resulta fundamental realizar una original atractiva, funcional e intuitiva propuesta técnica y visual.

ESe entiende por accesibilidad web a la utilidad con la que un mayor número de usuarios hacen de una plataforma, independientemente de sus conocimientos, capacidades físicas y de las características técnicas de sus dispositivos, es decir, toda persona debe lograr percibir, entender, navegar e interactuar dentro de un sitio web sin importar sus capacidades físicas, intelectuales y técnicas.

El Buscador Jurídico tiene como objetivo ser una web accesible, de tal forma que para transmitir la información jurídica y disminuir las barreras de comunicación e interacción con las cuales muchas personas se enfrentan, ha sido desarrollado y diseñado conforme a las recomendaciones y estándares del W3C; por ello se ha integrado una herramienta de accesibilidad para agregar opciones de configuración de pantalla y lectura de voz, con el objetivo de eliminar aquellas barreras que impiden a las personas acceder al universo de información que se publica en el BJ.

Justificación

Una web accesible puede beneficiar tanto a personas con discapacidades, para que logren percibir, comprender, navegar e interactuar con la Web, como a personas sin discapacidades pero que interactúan a través de diferentes dispositivos, o que presentan limitaciones de ubicación, problemas de conexión, discapacidad temporal o habilidades que cambian con la edad.

Elementos de accesibilidad

Las principales características que se han desarrollado en el BJ pretenden abarcar los siguientes elementos de accesibilidad:

Perceptible

El contenido y el diseño de todos los elementos del sitio web deben ser percibidos fácilmente a través de los sentidos y considerarse las limitantes físicas, intelectuales y técnicas, así como del entorno, para que el sitio web pueda ser accesible y ejecutable por todos los usuarios en cualquier momento y lugar.

Tamaño de letra

Una función implementada ha sido el ajuste del tamaño de letra, la cual permite incrementar el tamaño del texto para mejorar la legibilidad de la información.

Fuente

Una de las fuentes que forma parte de los estándares accesibles es la Arial, ya que aparte de ser una fuente común, es fácil de interpretar por todas las personas; otras de las fuentes incluidas en la herramienta son Calibri, Century Gothic, Helvetica, Tahoma y Verdana.

Dentro de las fuentes utilizadas en la herramienta de accesibilidad ha sido Lucida Brigth y Myriad Pro Regular, las cuales forman parte de los requerimientos de la imagen institucional de la SCJN y además cuentan con las características de legibilidad.

Alto contraste

La opción de color y contraste permite realizar cambios a negro sobre blanco y negro sobre amarillo, es decir, se modifican en su totalidad los colores originales de la plataforma. Esta funcionalidad no afecta la estructura del Buscador Jurídico, únicamente modifica los colores.

Operable

Es de suma importancia que las opciones que permitan el uso del mouse, pad y teclado estén disponibles para activar todas las funciones y acceder a toda la información.

Comportamiento de la web y navegación predecible

El BJ cuenta con la característica de navegación predecible, la cual permite al usuario conocer en qué módulo de la plataforma se encuentra ubicado y cómo actúan cada una de las funciones.

Con el uso constante de la plataforma, el usuario intuitivamente aprende a navegar en todas las páginas a través de enlaces y botones, utilizar los filtros de búsqueda y realizar consultas con la sección de ayuda y sugerencias de la plataforma.

Teclas de navegación y métodos abreviados

El BJ se ha codificado para ser compatible con el uso de teclas de navegación, las cuales se describen a continuación:

Nombre de la tecla ¿Cómo usarla?
Tabulador Presiona la tecla Tab (↹) para moverse entre distintos elementos seleccionables de texto o elementos de navegación como enlaces y botones.
Barra espaciadora Presiona la barra espaciadora para mover el cursor un espacio hacia delante.
Entrar Presionar la tecla Enter (↳) para ejecutar un comando escrito previamente.

En la tabla siguiente se muestran los métodos abreviados de teclado más útiles que también pueden usarse en el Buscador Jurídico.

Método abreviado ¿Cómo usarlo?
Flecha izquierda, Flecha derecha, Flecha arriba o Flecha abajo Mover el cursor un espacio o línea en la dirección de la flecha o desplazar una página web hacia abajo o arriba.
Inicio Mover el cursor hasta el final de una línea o ir al principio de una página web
Fin Mover el cursor hasta el final de una línea o ir al final de una página web
Ctrl + Inicio Ir al principio de un documento
Ctrl + Fin Ir al final de un documento
RePág Subir el cursor o la página una pantalla
AvPág Bajar el cursor o la página una pantalla
Ctrl + C Copiar el elemento seleccionado
Ctrl + A Seleccionar todos los elementos en un documento o una ventana
Alt + Flecha izquierda Permite volver a la página anterior
Alt + Flecha derecha Permite ir a la página siguiente

Es importante hacer mención de que cada una de estas combinaciones de teclas son compatibles con el sistema operativo Windows; en caso de usar un sistema operativo distinto, verificar si son compatibles.

Comprensible

La escritura debe ser entendible por todos los usuarios; en otras palabras, es importante contemplar el uso adecuado de las abreviaturas, modismos y neologismos para su fácil comprensión. Se debe buscar que el diseño, así como toda herramienta y funcionalidad sea predecible e intuitiva para una buena experiencia de navegación.

Compatible

Los sitios web deben ser compatibles con los navegadores y sistemas operativos, también con diferentes dispositivos y sus resoluciones de pantalla. Es fundamental seguir las recomendaciones, especificaciones técnicas, pautas y estándares de la Iniciativa de Accesibilidad Web del W3C (WAI) sobre este tema.

Diseño adaptativo a distintos dispositivos
